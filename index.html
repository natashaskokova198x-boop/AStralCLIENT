Часто задаваемые вопросы

Как установить Astral?

Распакуйте архив в любую папку. Киньте мод (.jar) в папку mods по пути майнкрафта скачайте версию фабрик 1.21.11 + фабрик Api

Могут ли меня забанить за использование чита?

Astral разработан с использованием передовых обходов античитов, но 100% гарантий нет. Для максимальной безопасности рекомендуем использовать легит функции

С какими версиями Minecraft совместим клиент?

Astral совместим с версиями только для 1.21.11

Как получить поддержку при возникновении проблем?

Присоединяйтесь к нашему Discord серверу или Telegram каналу для получения технической поддержки и обновлений.
